Jekyll -> Hugo

2018/05/03
  • Jekyll + servr体制からHugo + blogdown体制へ移行
    • blogdown時代なので時流に合わせて

引っかかりどころメモ

  • blogdown::build_site() -> docs以下に公開ファイル群を生成 -> git push -> github pagesで公開
  • blogdown::build_site()はデフォルトだと全Rmarkdownを再評価するので遅い
    • 少なくとも、chunk optionでcache=TRUEしておく
    • root/R/build.Rに生成用の関数を書いておいて、blogdown::build_site(method = "custom")
  • aboutなどのいくつかの単語は予約語で大文字小文字など問題が起こるのでmain menuには使わないほうが無難
